## Montar en desarrollo ‚öíÔ∏è

Para ejecutar este proyecto es necesario tener instalado nodejs, un gestor de paquete y el cli de angular.

Instalar dependencias
```bash
  npm install
```
Ejecutar aplicacion
```bash
  ng serve:o
```
